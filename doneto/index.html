Doneto

Um mini-framework CSS sem classe nenhuma!

Isso não faz sentido.
— ADRIANO, JAPA e RAK, sobre o Doneto.

O Doneto é um conjunto de estilos CSS com um conceito diferente, ele acredita que as tags html não necessitam de classes para serem estilosas. Sendo assim, todos os estilos CSS são aplicados diretamente nas tags. Para utilizá-lo você deve garantir apenas que os elementos de sua página são HTML5 e, "Doneto", eles já estão pronto para serem utilizados.

Não acredita? Então aperte ctrl + u e veja você mesmo!

Por quê eu escolheria o Doneto?

É claro, existem muitos outros frameworks front-end, então por que usar o Doneto?

Os demais frameworks, como o Bootstrap, são muito massivos para pequenas aplicações e são incrivelmente intrusivos. Além disso, é quase certo que você irá precisar fazer suas próprias classes para complementar o visual de sua página.

Você não deveria ter de alterar o seu código HTML simplesmente para ele funcionar em um framework. Prezando essa facilidade, surgiu o Doneto e sua maneira de tratar os estilos.

É claro, isso limita o seu alcance, mas é de grande ajuda para pequenos projetos ou como o pontapé inicial de grandes aplicações. Afinal, você iria fazer as suas próprias classe de qualquer maneira, não iria?

Como utilizar o Doneto?

Para utilizar o Doneto tudo o que você precisa fazer é usar o seu arquivo de css, o seu cabeçalho deve ficar parecido com isso: 

<!DOCTYPE html>
<html>
    <head>
        <link rel="stylesheet" href="https://samukeras.github.io/doneto/doneto.min.css"/>
    </head>
</html>
